# KABADDI REF GitHub Pages Upload

## 1. Create the repository

1. Open https://github.com/new
2. Repository name: `kabaddi-referee`
3. Public
4. Add a README file: ON
5. Create repository

## 2. Upload files

1. Open the new `kabaddi-referee` repository.
2. Click `Add file`.
3. Click `Upload files`.
4. Drag all files in this folder into GitHub.
5. Click `Commit changes`.

Upload these files:

- `.nojekyll`
- `app.js`
- `icon.svg`
- `index.html`
- `manifest.json`
- `service-worker.js`
- `styles.css`

`UPLOAD_STEPS.md` is only this instruction file, so uploading it is optional.

## 3. Turn on GitHub Pages

1. Open `Settings`.
2. Open `Pages`.
3. Source: `Deploy from a branch`.
4. Branch: `main`.
5. Folder: `/ (root)`.
6. Click `Save`.

After a few minutes, the app URL will be:

```text
https://YOUR-GITHUB-USER-NAME.github.io/kabaddi-referee/
```

Open that URL on iPhone Safari, then use Share -> Add to Home Screen.
